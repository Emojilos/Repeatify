TASK-017 DONE — SRS engine (modified SM-2 algorithm with exam countdown factor)

- ReviewResult dataclass: next_review_date, new_interval, new_ease_factor
- calculate_next_review(card, self_assessment, exam_date, today) — core SM-2 logic:
  - again → interval=1, ef-=0.2
  - hard → interval*=1.2, ef-=0.15
  - good → interval*=ef
  - easy → interval*=ef*1.3, ef+=0.15
- ease_factor clamped at minimum 1.3
- Exam countdown factor: <14d→60%, <30d→75%, <90d→100%, >=90d→120%
- Minimum interval always >= 1 day

  - All 4 self_assessment variants (again/hard/good/easy)
  - All 4 exam countdown ranges + boundary values
  - EF floor enforcement (single + series of again)
  - Interval compression with exam proximity
  - Minimum interval clamping
  - Invalid assessment error handling
